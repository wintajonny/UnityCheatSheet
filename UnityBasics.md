# Unity Basics

## Scene Shortcuts
- Q - View Tool
- W - Move Tool
- E - Rotate Tool
- R - Scale Tool
- T - Rect Tool --> Move/Rotate/Scale UI Elements
- Y - Transform Tool --> Move + Rotate + Scale

## Scene Camera Movement
- ALT + LeftClick -> Revolve around Object/Center
- RightClick + WASD -> Flythrough Mode 
  - Q - Down
  - E - Up

## Put Camera (Game Object) to position of 'Developer' 
- Select Camera
- Ctrl + Shift + F
