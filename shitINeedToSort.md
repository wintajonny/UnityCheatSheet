## Scene Shortcuts
- Q - View Tool
- W - Move Tool
- E - Rotate Tool
- R - Scale Tool
- T - Rect Tool --> Move/Rotate/Scale UI Elements
- Y - Transform Tool --> Move + Rotate + Scale

## Flythrough Mode 
- Q - Down
- E - Up


## Put Game Object to position of 'Developer' 
- Select GameObject (e.g. Camera)
- Ctrl + Shift + F


## Playmode Tint 
Edit -> Preferences -> Colors -> Playmode Tint
